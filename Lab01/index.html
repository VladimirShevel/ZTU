Державний Університет "Житомирька політехніка"

2 вересня на пленарному засіданні Верховної Ради України було ухвалено постанову про те, щоб відсвяткувати на державному рівні сотий ювілей від дня заснування Державного університету "Житомирська політехніка".[1]

У вересні 2020 року Державний університет "Житомирська політехніка" разом з представниками норвезького уряду підписали угоду про реабілітацію учасників АТО/ООС[2]

29 липня 2021 року у Державному університеті "Житомирська політехніка" створили інтерактивний музей науки і технологій "Політехнік".[3]

[1] - 100-річчя Житомирської політехніки будуть відзначати на державному рівні. zhytomyr.one (укр). Процитовано 21 вересня 2020.

[2] - Житомирська політехніка підписала угоду з Норвегією. zhytomyr.one (укр). Процитовано 28 вересня 2020.

[3] - На базі "Житомирської політехніки" створили інтерактивний музей науки і технологій - zhytomyr-furute.com.ua.(укр). Процитовано 3 листопада 2021.

Завдання 2

1. Знак +

Визначення: a+b означає "a додати b"

Приклад: 2+2

2. Знак −

Визначення: a−b означає "a відняти b"

Приклад: 2−2

3. Знак ×

Визначення: a×b означає "a помножити на b"

Приклад: 2×2

4. Знак ÷

Визначення: a÷b означає "a поділити на b"

Приклад: 2÷2

5. Знак §

Визначення: §1 означає "початок параграфу 1"

Приклад: §1

Chapter 1: Introduction to HTML

What is HTML?

HTML (Hypertext Markup Language) is a markup language used to structure and format content on the web. It uses a system of tags to define the different elements of a web page, such as headings, paragraphs, and links.

Для того щоб створити таку сторінку, введіть наступний код:


<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <title>VSShevel</>
</head>
<body>
    <h3>HTML Tutorial</h3>
    <p>Welcome to our HTML tutorial! In this tutorial, you will learn the basics of HTML and how to use it to create
        your own web pages.</p>

    <h4>Chapter 1: Introduction to HTML</h4>
    <article>
        <h5>What is HTML?</h5>
        <p>HTML (Hypertext Markup Language) is a markup language used to structure and format content on the web. It
            uses a system of tags to define the different elements of a web page, such as headings, paragraphs, and
            links.</p>
    </article>
</body>
